# コンセプト概要

## 一言で
**「3歳の子どもが“自分のウサギ”と会話しながら育てる」**ことを核にした、会話型ペットWebアプリ（PWA） + 親向けの言語発達ログ閲覧サイト。

## 背景 / ねらい
- 娘がタマゴッチにハマっているが、既存のゲームは基本的に **同じループの繰り返し**で飽きる。
- 子どもは今まさに語彙・言い回し・概念を爆速で獲得する時期。
- LLM を裏側に置くと、**子どもの成長に合わせてペット側も成長する**体験が作れる。
- ついでに（これが実は一番面白い）、会話ログから **言語獲得の軌跡**が見える。

## 体験の中心（ユーザー体験）
### 子ども側（メイン画面）
- 画面にウサギがいて、**話しかけると返事をする**
- 遊び（追いかけっこ等）の感想や相槌を自然に言う
- 夜などに「そろそろお風呂にしよっか？」のように **生活習慣を提案**（説教しない）
- 子どもの語彙が増えるにつれ、ウサギの話し方や遊び方も変わる（進化）

### 親側（ダッシュボード）
- どんな言葉を覚えたか
- どんな言い回しが増えたか
- 新しい概念（反省/交渉/相手の気持ち等）が見えたタイミング
- 会話のハイライト（週次/月次）

## 絶対に外さない設計方針
1. **音声は必ず保存**  
   音声認識精度は今後上がる。未来に再解析できるように raw audio を残す。
2. **“私のウサギ感”**  
   口調/記憶/成長の一貫性がないと、子どもはすぐ冷める。
3. **繰り返しの価値**  
   同じフレーズは TTS を一度生成して保存 → 再生。速度・安定性・コストが良くなる。
4. **安全（少なくとも事故らない）**  
   3歳に LLM を直結するのはリスク。最低限の安全なプロンプト・拒否・保護者介入導線を入れる。

## 非ゴール（最初はやらない）
- ストアで売る / 課金
- 多言語 / 多ユーザー
- 形式張ったプライバシー/コンプラ（ただし公開するなら話が変わる）
