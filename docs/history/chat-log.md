## アシスタント（要点の要約）
- Web Speech API（SpeechRecognition）はブラウザ組み込みの音声認識で、対応ブラウザが限定的。
- 互換性と将来の再解析のため、MediaRecorderで raw audio を保存してサーバ側ASRに切替できる設計が現実的。
- Cloudflare（Workers + D1 + R2 + Queues + Durable Objects）で一体運用するのが楽。
